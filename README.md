
# 📸 اینستاگرام دانلودر (ربات تلگرام)

رباتی ساده و سریع برای دانلود **ویدیو، عکس، ریلز و استوری** از اینستاگرام، مستقیماً داخل تلگرام.

[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://python.org)
[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://core.telegram.org/bots)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

---

## ✨ قابلیت‌ها
- دانلود پست‌های عمومی اینستاگرام (عکس و ویدیو)
- دانلود ریلز و استوری
- دانلود از هایلایت (اگر لینک مستقیم داشته باشید)
- عدم نیاز به لاگین در اینستاگرام
- حذف خودکار فایل‌های موقت برای حفظ حریم خصوصی

---

## 🚀 نحوه استفاده (برای کاربر نهایی)
1. ربات را در تلگرام پیدا کنید (لینک ربات خود را اینجا بگذارید)
2. دستور `/start` را بزنید
3. لینک پست/استوری/ریلز اینستاگرام را برای ربات بفرستید
4. چند ثانیه صبر کنید، فایل دانلود شده برایتان ارسال می‌شود

---

## 🛠️ نصب و اجرا (برای توسعه‌دهندگان)

### پیش‌نیازها
- Python 3.8+
- توکن ربات تلگرام (از [@BotFather](https://t.me/BotFather) بگیرید)

### مراحل نصب

```bash
git clone https://github.com/yourusername/instagram_downloader_telegram_bot.git
cd instagram_downloader_telegram_bot

# نصب کتابخانه‌ها
pip install -r requirements.txt

# تنظیم توکن
cp .env.example .env
# سپس فایل .env را ویرایش کرده و توکن واقعی را وارد کنید

# اجرای ربات
python bot.py
```

> 💡 **نکته**: برای اجرای همیشگی روی سرور می‌توانید از systemd یا screen استفاده کنید.
